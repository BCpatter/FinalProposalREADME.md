My project recreates the emission spectrum of a violet laser diode at different current values using numerical modeling. I collected experimental spectra and fit them with Gaussian curves to get values like peak intensity, center wavelength, linewidth, and background offset. Those fitted parameters were then interpolated using SciPy so the code can predict and generate realistic spectra at almost any current value.

In order to run my code, go to the folder that is titles "Code to Simulate" and go to the "Simulation Code.ipynb". Here you'll find the code in its entirety. In order to run it just press the run button and choose the Python Kernel. Then, a box will show up at the top asking you to put an input. Enter a number between 0-28 (can go higherbut my data stops at 28). This value that you are inputting is the input current (in mA) for the laser. The higher the input, the taller and skinnier the graph will be. At the bottom of the simulated graph, you'll also find the peak intensity value and the FWHM (Full Width Half Maximum) value for the simulated curve. In addition to this, in the README section, there is a methods explanation file. This file explains the equations and what they are used for in the code.
